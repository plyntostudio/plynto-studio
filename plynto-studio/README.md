# Plynto Studio — Landing Page

A clean, bold landing page for Plynto Studio. Built with plain HTML, CSS, and JS — no frameworks, no build step needed.

## Deploy to Vercel (2 minutes)

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Click **Deploy** — that's it. Vercel auto-detects static HTML.

## Project structure

```
plynto-studio/
├── index.html     # Main page
├── style.css      # All styles
├── main.js        # Scroll reveal + copy number
├── vercel.json    # Vercel routing config
└── README.md
```

## Customising

- **WhatsApp number**: Search for `8134877412` in `index.html` and replace with your number
- **Services**: Edit the `.service-card` blocks in `index.html`
- **Colors**: Change `--accent` and `--green` in `style.css` `:root`
- **Fonts**: Swap `Syne` / `DM Sans` in the Google Fonts link in `index.html`
